tacticofthetoe is an interactive web application designed for only one purpose; to challenge you with a thrilling game of tictactoe with amazing retro sound effects and animations! Hmph, well ..it's tictactoe basically.


This app has been built mostly using Bootstrap and Flask. Bootstrap is an open-source repository for front-end web development and Flask is a so-called "micro-framework" for building web apps using Python. Other tools used are JavaScript for ajax and virtualenv for packaging. The game itself is simple, it provides you with an option between playing against an AI or another human (if you have one close by). The game can then be played and eventually the results are displayed.


Enjoy! :)
